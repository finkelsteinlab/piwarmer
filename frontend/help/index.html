Bugs and New Features
Please report bugs or request new features using the issue tracker on Github.
Contributing
Raspberry PID is a free and open source project and we welcome contributions from anyone. You can make a pull request at any time, though if you'd like to discuss it first, you can open an issue with just a proposal.
Acknowledgments
